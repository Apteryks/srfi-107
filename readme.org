* SRFI 107: XML reader syntax

** by Per Bothner

This repository hosts [[http://srfi.schemers.org/srfi-107/][SRFI 107]]: XML reader syntax, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

This SRFI is in /final/ status.

The full documentation for this SRFI can be found in the [[http://srfi.schemers.org/srfi-107/srfi-107.html][SRFI Document]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-107/][join the SRFI-107 mailing list]] and send your message there.
